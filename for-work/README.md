# StyleMart — E-Commerce Platform

> Modern, full-stack e-commerce web application — Amazon / Noon inspired storefront with a complete admin panel.

---

## 🇸🇦 الوصف بالعربية

**StyleMart** هو متجر إلكتروني متكامل تم تطويره بأحدث التقنيات، مستوحى من تجربة كبرى المنصات العالمية مثل Amazon و Noon. يتميز بواجهة عصرية، أداء سريع، وتجربة مستخدم سلسة على جميع الأجهزة.

### المميزات الرئيسية
- 🛍️ **تصفح ذكي للمنتجات**: تصنيفات، فلاتر، بحث فوري، فرز حسب السعر والتقييم
- 🛒 **سلة تسوق متقدمة** مع حفظ تلقائي، أكواد خصم، وقائمة المفضلة
- 👤 **حسابات المستخدمين**: تسجيل دخول عبر البريد أو Google، إدارة الطلبات والعناوين
- 💳 **نظام دفع كامل** مع ملخص الطلب، شحن، ضرائب، وأكواد الخصم
- 🔐 **لوحة تحكم للإدارة**: إدارة المنتجات، الطلبات، المستخدمين، والتقارير
- 📱 **تصميم متجاوب 100%** يعمل بسلاسة على الجوال والتابلت والكمبيوتر
- ⚡ **أداء عالي** عبر تحميل كسول (Lazy Loading) وتقسيم الكود
- 🔍 **SEO محسّن** مع بيانات منظمة (Schema.org) و Open Graph
- 🎨 **هوية بصرية موحدة** ونظام تصميم متكامل (Design System)

### التقنيات المستخدمة
- **الواجهة الأمامية**: React 18, Vite 5, TypeScript, Tailwind CSS, shadcn/ui, Framer Motion
- **إدارة الحالة**: TanStack Query, React Context
- **الخلفية**: Supabase (PostgreSQL, Auth, RLS, Edge Functions)
- **الأمان**: Row-Level Security، نظام أدوار آمن (`has_role`)
- **التوجيه**: React Router v6 مع Code Splitting

## 🇬🇧 English Description

**StyleMart** is a production-grade e-commerce platform inspired by Amazon and Noon. Built with a modern stack, it delivers a fast, polished shopping experience across mobile, tablet, and desktop — with a full admin back office.

### Key Features
- 🛍️ **Smart product browsing** — categories, filters, live search, sorting by price/rating
- 🛒 **Advanced cart** — persistent storage, promo codes, wishlist, save-for-later
- 👤 **User accounts** — email + Google OAuth, order history, address book
- 💳 **Full checkout** — order summary, shipping, tax, promo-code discounts
- 🔐 **Admin dashboard** — product, order, and user management with role-based access
- 📱 **Fully responsive** — pixel-perfect on phone, tablet, and desktop
- ⚡ **High performance** — route-based code splitting, lazy loading, optimized images
- 🔍 **SEO optimized** — JSON-LD structured data, OpenGraph, Twitter cards, sitemap
- 🎨 **Cohesive design system** — semantic HSL tokens, consistent components

### Tech Stack
- **Frontend**: React 18, Vite 5, TypeScript, Tailwind CSS, shadcn/ui, Framer Motion
- **State**: TanStack Query, React Context API
- **Backend**: Supabase (PostgreSQL, Auth, Row-Level Security, Edge Functions)
- **Security**: RLS policies, secure role checks via `has_role()` SECURITY DEFINER function
- **Routing**: React Router v6 with lazy-loaded routes
